# NOSTRA – Fashion E-commerce Website

NOSTRA is a responsive fashion e-commerce website developed using **HTML5**, **CSS3**, and **JavaScript**. The project features a clean and modern user interface inspired by fashion shopping websites and is designed to provide a smooth browsing experience across desktop, tablet, and mobile devices.

## Features

* Responsive design for all screen sizes
* Modern navigation bar
* Hero banner section
* Product categories
* New Arrivals section
* Most Wanted products
* Stylish product cards
* Newsletter subscription section
* Footer with company information
* Clean and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Learning Outcomes

* Responsive web design
* CSS Flexbox and Grid layouts
* Modern UI development
* JavaScript DOM manipulation
* Website structure and component organization

This project was created as part of my frontend web development practice to improve responsive design and user interface development skills.
